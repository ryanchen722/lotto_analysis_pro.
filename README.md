🔭 為什麼選擇高斯模型？

現代人工智慧（AI）的基石源於高斯的誤差理論。本專案將這種「科學預測」的精神引入彩券分析，利用以下 AI 底層數學邏輯進行運算：

常態分佈過濾 (Gaussian Filtering)：
利用高斯分佈的 $\mu$ (均值) 與 $\sigma$ (標準差)，排除 95.4% 以外的統計異常組合，確保號碼分佈處於機率最穩定的黃金區間。

最小誤差優化 (Least Squares Logic)：
在 8000 次模擬中，篩選出與歷史頻率權重偏移量最小的組合，模擬 AI 訓練中的誤差最小化過程。

數論均衡 (Number Theory)：
繼承高斯在《算術研究》中的同餘思想，確保組合在 Modular 空間中的分佈均勻。

🚀 專案亮點

整合型架構：一個程式同時支援「大樂透」與「539」。

動態參數校正：根據上傳的 Excel 數據自動生成統計模型。

蒙地卡羅模擬：結合 8,000 次高速運算的魯棒性分析。
